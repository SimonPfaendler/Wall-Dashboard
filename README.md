# Wall Dashboard

A clean, dark-mode dashboard designed for an always-on tablet display.

## Features
- **Clock**: Large digital time and date.
- **Weather**: Current conditions and daily high/low.
- **Trash Schedule**: Next pickup indicator (requires `.ics` file).

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Add Trash Schedule:**
    Place your calendar file at `public/trash.ics`.

3.  **Run locally:**
    ```bash
    npm run dev
    ```

## Technology
- React + Vite
- TailwindCSS
